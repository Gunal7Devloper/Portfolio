Given an integer n, return the nth Recamán's term. ------------------------------------------------------------------------------ Handle base case If n < 1, return 0 immediately since a(0) = 0. Initialize the sequence Start with an array containing only the first value of the Recamán sequence, which is 0. Iterate from 1 to n For each index i from 1 to n, calculate the next term in the sequence. Get the previous term Retrieve the most recent value from the sequence (i.e., value at index i - 1). Try subtracting the index Subtract the current index i from the previous value to get a candidate value. Check if subtraction result is valid If the result is negative or already exists in the sequence, discard it. Fallback to addition If subtraction was invalid, compute the new value by adding the current index i to the previous value. Add the correct next value to the sequence Append the valid result (either the subtraction or addition result) to the sequence. Repeat until the nth value is computed Continue this process for all values up to the nth index. Return the nth value After the loop ends, return the value at index n of the sequence. ------------------------------------------------------------------------

Write a function to find the nth term of Recamán's sequence.

Instructions Recamán's sequence is defined as follows:
1. a(0) = 0; for n > 0.
2. a(n) = a(n - 1) - n if that number is positive and not already in the sequence.
3. Otherwise, a(n) = a(n - 1) + n.
The first few terms of the sequence are:
1. r(0) = 0.
2. r(1) = r(0) - 1 is not positive, so r(1) = r(0) + 1 = 1.
3. r(2) = r(1) - 2 = -1 is not positive, so r(2) = r(1) + 2 = 3.
4. r(3) = r(2) - 3 = 0 has been used, so r(3) = r(2) + 3 = 6.
5. r(4) = r(3) - 4 = 2.
6. r(5) = r(4) - 5 = -3 is not positive, so r(5) = r(4) + 5 = 7.
So, Recamán's sequence is: 0, 1, 3, 6, 2, 7, ...
Input:
s = "{ [ ( ) ] }"
Output:
true
Approach:
  1. Initialize an empty stack.
  2. Define matching pairs for closing brackets.
  3. Iterate through each character in the input string.
  4. If the character is an opening bracket, push it onto the stack.
  5. If the character is a closing bracket, check the top of the stack.
  6. If the top matches the corresponding opening bracket, pop it from the stack.
  7. If it doesn't match or the stack is empty, return false.
  8. After iteration, return true if the stack is empty (all brackets matched).
