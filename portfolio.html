Write a function to check if the input string has valid parentheses.

Instructions A string has valid parentheses only when every opening bracket is closed in the correct order. Examples of some string with valid parentheses are:
1. "( )" - each opening bracket ( has a corresponding closing bracket )
2. "( )[ ]" - each opening bracket ( and [ has a corresponding closing bracket ) and ]
3. "( ( [ ] ) { } )" - each type of bracket is correctly closed in order.
Examples of some string with invalid parentheses are:
1. ")(" - the closing bracket ) appears before the opening bracket (
2. "( { [ } ) ]" - brackets are closed in wrong order.
If the string has valid parentheses, return true. Otherwise, return false.
Input:
s = "{ [ ( ) ] }"
Output:
true
Approach:
  1. Initialize an empty stack.
  2. Define matching pairs for closing brackets.
  3. Iterate through each character in the input string.
  4. If the character is an opening bracket, push it onto the stack.
  5. If the character is a closing bracket, check the top of the stack.
  6. If the top matches the corresponding opening bracket, pop it from the stack.
  7. If it doesn't match or the stack is empty, return false.
  8. After iteration, return true if the stack is empty (all brackets matched).
